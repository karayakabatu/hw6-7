# Quiz App – Vibe Coding Project

This project is a simple quiz application created as part of an academic assignment focused on vibe coding and AI-assisted software development. The main purpose of the project is to explore how AI-first development tools can support the software development process beyond basic code completion, including structuring logic, refining functionality, and improving usability.

The project was developed using Cursor, an AI-powered code editor, with an emphasis on clean structure, clear logic, and user-friendly interaction.

# Features

User name input before starting the quiz
Multiple-choice quiz questions
Shuffled questions and answer options on each restart
Score tracking
Restartable quiz flow
Top 5 leaderboard stored using browser localStorage
Responsive layout for desktop, tablet, and mobile screens
Simple and clean user interface

# Technologies Used

HTML
CSS
JavaScript
Cursor (AI-powered code editor)

# Vibe Coding Tool

Cursor was used throughout the development process to generate the initial project structure, assist with logic implementation, iteratively add features, and improve the overall user interface and user experience.

# How to Run (Without Docker)

The application runs entirely on the client side. Opening the index.html file in any modern web browser is sufficient to use the application.

# Run with Docker

This project can also be run using Docker to provide a consistent and platform-independent execution environment.

Open a terminal in the project root directory.

Run the following command:

docker compose up --build

Open your web browser and navigate to:

http://localhost:8080

The application will be running and accessible through the browser.

To stop the application, press Ctrl + C in the terminal and then run:

docker compose down
