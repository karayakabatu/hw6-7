This project is a simple quiz application created for an assignment about vibe coding and AI-assisted development. The main purpose of the project is to understand how AI-first tools can support the development process beyond basic code completion.

The project was built using Cursor, an AI-powered code editor, and focuses on functionality, clean structure, and usability.

---

## Features

* User name input before starting the quiz
* Multiple-choice quiz questions
* Questions and answer options are shuffled on each restart
* Score tracking
* Restartable quiz
* Top 5 leaderboard stored using localStorage
* Responsive layout for desktop, tablet, and mobile screens
* Simple and clean user interface

---

## Technologies Used

* HTML
* CSS
* JavaScript
* Cursor (AI-powered code editor)

---

## Vibe Coding Tool

Cursor was used throughout the development process to generate the initial structure, fix logic issues, add features iteratively, and improve the user interface.

---

## How to Run

The application runs entirely on the client side. Opening the index.html file in any modern web browser is sufficient.
